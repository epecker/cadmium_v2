Detailed Description

template<typename S, typename V>
class cadmium::celldevs::GridCell< S, V >

Abstract base class for cells in grid Cell-DEVS scenarios.

Template Parameters
Stype used to represent cell states.
Vtype used to represent vicinities between cells.
